# Aureline Architecture Studio

Aureline Architecture Studio is a premium architectural landing page built for modern residences, custom villas, and project showcases. The site presents services, selected works, process steps, testimonials, and a contact section in a clean, elegant layout.

## Live Demo

[View the website](https://tharindu714.github.io/Aureline.com/)

## Overview

This project is designed to help an architecture or interior design brand present its services professionally and capture inquiries from potential clients.

### Highlights

- Premium visual presentation
- Clear service breakdown
- Featured project showcase
- Simple process workflow
- Client testimonial section
- Strong consultation / inquiry call-to-action

## Site Sections

- **Hero section** — brand introduction and consultation CTA
- **Services** — house design, 3D visualization, council drawings, and technical packages
- **Selected Works** — example projects such as *Casa Ember*, *Marina Crest Villa*, *Studio Noir*, and *Oakline Residence*
- **Process** — discovery, concept design, technical development, and delivery
- **Testimonials** — client feedback for credibility
- **Contact** — inquiry section for new leads

## Content Notes

The current version is set up as a sample and can be customized with:

- brand logo

- service pricing
- contact details

3. The repository is published through **Settings → Pages**.
